Introduction CSS

CSS est l'abréviation de « Cascading Style Sheets ».C'est un langage de style qui définit la présentation des documents HTML. Par exemple, CSS couvre les polices, les couleurs, les marges, les lignes, la hauteur, la largeur, les images d'arrière-plan, les positionnements évolués et bien d'autres choses. Attendez de voir !


Comment ça marche ?

Le CSS est très facile d'utilisation on peut l'utilisé directement dans une page html, avec la fonction :

style=""

par exemple si on veut écrire un titre de couleur rouge, on fera

h1 style="color:red;"

Utilisation plus courante

Bien sur si la totalité du CSS était écrit sur une page HTML, cela donnerait des pages bien trop charger et très peut lisible c'est pourquoi dansla plus part des cas on écrit le CSS sur une page a part et on la relis vie un lien placé dans la tête de notre page html

link rel="stylesheet "href="pageCSS"

ce faisant notre code reste facilement accessible et lisible, on peut aussi le noté dans la partit "head" de notre page html au lieu d'u mettre un lien vers une autre pages, mais un fichier CSS fait rapidement pas mal de ligne et ça sera toujours plus lisible de le traiter a part.


Des exemple concrêt ?

  • Définir la couleur d'un élément, sur du texte ou un titre ce sont les caractère qui prendrons la couleur définit

    • p{color:#FF0000;}

  • Pour changer le fond d'un élément, ça peut être le fond d'une page entière ou d'une partit de la page, voir juste d'un mot pour le mettre en valeur

    body{background-color:#8888AA;}

  • Il est possible de définir la hauteur ou la largeur d'un élément, très pratique surtout avec les image pour partager des image qui n'on pas forcément totue les même dimenssion mais en conservant une mise en page cohérente

    height: 80px
    width: 75%

    Vous pouvez utilisez comme ordre de grandeur les px, les %, les em, les rem, et d'autre encore...

  • N'oublier pas vous pouvez cibler des élément tel que la totalité de vos paragraphe, de vos div ou de vos titre, mais il sera en général plus pratique d'associé vos élément a des "id" ou des "class" et d'attribué les propriété du CSS a ses élément que vous pourez alors ciblez avec plus de précision

  • Pour les couleurs, vous pouvez tout simplement les écrire, ou utilisé un code RGB

    color: rgb(200,0,100)

    Mais aussi l'écrire en exadécimale, la solution que je préfère, ça fait un peut peur au début de remplacé les dizaine par des seizaine et d'inclure des lettre dans nos numéro mais une fois qu'on a compris c'est hyper simple ! color: #AAAA30;
